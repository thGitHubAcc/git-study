#GIT命令

## 查看git版本、
```
git --version
```
## 清屏
```
clear
```

## 设置用户名和邮箱
```
git config --global user.name "fth"
git config --global user.email "364870745@qq.com"
```

## 初始化仓库
```
git init(.git目录是隐藏的)
```

## 查看目录
```
ll(查看目录)
ll -la(查看所有目录)
```
